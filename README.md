# Fitdata Portal

A fitness data visualization portal that displays steps and heart rate metrics with interactive charts and configurable
date ranges.

## Features

- **Steps Tracking**: View daily step counts with multiple time range options
- **Heart Rate Monitoring**: Display resting heart rate and heart rate zones
- **Interactive Charts**: Visualize fitness data using responsive line charts
- **Flexible Date Ranges**: Support for multiple viewing periods (7, 14, 30 days, current week)
- **Real-time Data**: Fetch and display fitness metrics from backend API

## Tech Stack

### Frontend

- **React** with TypeScript
- **Recharts** for data visualization
- **TailwindCSS** for styling
- **Vite** as build tool

### API Integration

- RESTful API endpoints for steps and heart rate data
- Type-safe API client with TypeScript
